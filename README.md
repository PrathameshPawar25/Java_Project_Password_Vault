# Local File-Based Password Vault (Pure Java)

This project implements an offline, encrypted password vault using only core Java.

## Features
- AES-256 encryption
- Secure SHA-256 hashed master password
- Add, view, and store credentials safely
- Encrypted vault stored locally as vault.dat
- Fully offline, no frameworks
- Command Line Interface

## Usage
javac *.java
java Main

The program will guide you through creating or opening a vault.
